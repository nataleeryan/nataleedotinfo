Crewtrip: The Group Vacation Planner


Overview

Planning a trip can be burdensome and increase in difficulty as the travel group size grows. From trying to coordinate availability to deciding what kind of excursion the group actually wants to go on, the whole process could benefit from some streamlining. We created an application and API that allows groups collaboratively decide trip itinerary details in an interactive and fun manner. I was the frontend developer and UX designer for this project on a team of 3.
